# AgriGuard Jordan (2026) - Agri-FinTech AI Agent 🚀

**مستشار المزارع الذكي** - نظام متكامل مؤتمت للتأمين الزراعي والشمول المالي في الأردن، يربط بين بيانات دائرة الأراضي والمساحة، وزارة الزراعة، وتحليلات الأقمار الصناعية.

## 🌟 الرؤية العامة (The Vision)
يهدف المشروع إلى أتمتة عمليات التأمين الزراعي في الأردن لعام 2026، من خلال استخدام الذكاء الاصطناعي (AI Agent) لتحليل المخاطر الجغرافية والمناخية وتوفير باقات تأمين مخصصة مدعومة حكومياً لصغار المزارعين.

---

## 🛠 التقنيات المستخدمة (Tech Stack)
*   **Frontend:** React.js + TypeScript + Tailwind CSS.
*   **Charts:** Recharts (لتمثيل البيانات المناخية وNDVI).
*   **Icons:** Lucide React.
*   **AI Engine:** Simulated Gemini 1.5 Pro (مع إمكانية الربط المباشر).
*   **APIs:** OpenWeatherMap API & Planet Insights API.

---

## 🚀 تعليمات التشغيل (Setup Instructions)

تأكد من تثبيت [Node.js](https://nodejs.org/) على جهازك، ثم اتبع الخطوات التالية:

1. **تثبيت الاعتمادات:**
   ```bash
   npm install
   ```

2. **تشغيل المشروع في وضع التطوير:**
   ```bash
   npm run dev
   ```

3. **فتح المتصفح:**
   انتقل إلى الرابط `http://localhost:5173` لمشاهدة المنصة.

---

## 📋 ميزات النظام (Core Features)

1.  **هوية الأرض (Identity Layer):** تحقق قانوني يحاكي الربط مع دائرة الأراضي والمساحة الأردنية عبر رقم القطعة والحوض.
2.  **تحليل AI Agent المؤتمت:** نظام ذكي يحلل حالة الأرض والطقس ويقدم تقريراً فنياً فورياً.
3.  **بيانات حية (Real-time Data):** جلب درجات الحرارة والرطوبة الحقيقية ومؤشرات NDVI بناءً على الإحداثيات الجغرافية.
4.  **باقات التأمين الرسمية:** عرض باقات تأمين متوافقة مع صندوق إدارة المخاطر الزراعية بوزارة الزراعة (موسم 2026).
5.  **الشمول المالي:** بوابة دفع محاكاة تدعم المحافظ الإلكترونية المحلية (Zain Cash / Orange Money).

---

## 🗺 حالات الاختبار الحقيقية (Use Cases)
يدعم النظام 3 نماذج لأراضٍ حقيقية لإظهار التباين الجغرافي والمناخي في الأردن:
*   **المفرق (حوض الصبحة):** التركيز على مخاطر الجفاف ونقص الأمطار.
*   **دير علا (الأغوار الوسطى):** التركيز على مخاطر الصقيع الإشعاعي وصحة المحصول.
*   **إربد (بني كنانة):** التركيز على الإنتاجية العالية ومخاطر الآفات وانجراف التربة.

*   **الملكية:** دائرة الأراضي والمساحة (DLS).
